# Yes Bank Stock Price - Exploratory Data Analysis (EDA)

## 📋 Project Overview
This repository contains the **Exploratory Data Analysis (EDA)** of Yes Bank's monthly stock price data from July 2005 to November 2020.

### 🎯 Objective
To understand historical stock price patterns, trends, volatility, and relationships between Open, High, Low, and Close prices through visualizations and statistical analysis.

## 📊 Key Insights
- Strong upward trend from 2005 to 2018
- Sharp decline in 2020 due to liquidity crisis
- Very high correlation between Open, High, Low, and Close prices (> 0.98)
- Increased volatility after 2018
- Right-skewed distribution of closing prices

## 🔍 Analysis Performed
- **Univariate Analysis**: Distribution, Boxplots, Time Series
- **Bivariate Analysis**: Scatter plots, Year-wise trends
- **Multivariate Analysis**: Correlation Heatmap, Pair Plot, Candlestick Chart
- **Feature Engineering**: HL_Pct, OC_Pct, Rolling Statistics

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly

## 📁 Files
- `Yes_Bank_EDA.ipynb` → Main EDA Notebook
- `data_YesBank_StockPrices.csv`

## 📈 Visualizations Included
- 20+ charts following UBM rule (Univariate, Bivariate, Multivariate)
- Time series trends, volatility analysis, outlier detection, etc.
